BotCompRemote

A Python Discord bot that lets you control your computer remotely. It offers features such as retrieving public and local IP addresses, shutting down your computer, managing browser tabs, controlling the webcam, and even editing notes remotely!

  • Retrieve public and local IP addresses
  • Shut down the computer remotely
  • Control browser tabs
  • Perform Google searches from Discord
  • Take screenshots
  • Control the webcam feed
  • Remotely run or view applications
  • Create and edit notes via Notepad
